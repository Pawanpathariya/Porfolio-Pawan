# Pawan Pathariya - Full Stack Developer Portfolio

A professional React portfolio website showcasing experience at **Magnet Brains** (Pabbly) and **Inteliostack.ai**, education, skills, and projects.

**Frontend only** - Contact form sends emails to pawanpathariys@gmail.com via [FormSubmit](https://formsubmit.co).

## Tech Stack

- **Frontend:** React, Vite
- **Contact Form:** FormSubmit (form-to-email, no backend)

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build for Production

```bash
npm run build
npm run preview
```

## Customization

Edit `src/data/portfolioData.js` to update:

- Profile (name, email, LinkedIn, GitHub, phone)
- About summary and highlights
- Experience entries
- Education
- Skills
- Projects

## Contact Form

Uses [FormSubmit](https://formsubmit.co) - submissions go directly to **pawanpathariys@gmail.com**.

**First-time setup:** When someone submits the form for the first time, FormSubmit sends a confirmation email. Click the link to activate.
